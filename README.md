# movie-app

Personal Movie APP

## Run Locally

Clone the project

```bash
  git clone https://github.com/mattzbik/movie-app.git
```

Go to the project directory

```bash
  cd movie-app
```

Install Image

```bash
docker-compose build
```

Start Server

```bash
docker-compose up
```

Stop Server

```bash
docker-compose down
```

View GraphQL Interface Locally

```bash
http://localhost:4000/graphql
```

## Installation

Docker Commands
