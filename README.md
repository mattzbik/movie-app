# movie-app

Personal Movie APP

## Run Locally

Clone the project

```bash
  git clone https://github.com/mattzbik/movie-api.git
```

Go to the project directory

```bash
  cd movie-api
```

Install Image

```bash
docker-compose build
```

Start Server

```bash
docker-compose up
```

Stop Server

```bash
docker-compose down
```

View Application Locally

```bash
http://localhost:3000/
```
